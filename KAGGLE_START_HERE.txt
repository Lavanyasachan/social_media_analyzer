================================================================================
                       WHAT THE SCRIPT DOES
================================================================================

The script is COMPLETELY SELF-CONTAINED and will:

✅ Load the fabric defects dataset
✅ Create training and validation splits
✅ Build a ResNet50 CNN model
✅ Train the model (with GPU if enabled)
✅ Evaluate accuracy and create confusion matrix
✅ Generate beautiful visualizations
✅ Save the trained model
✅ Save all plots and metrics

NO manual steps needed! Just run and wait.

================================================================================
                      ENABLE GPU (RECOMMENDED!)
================================================================================

Before running, enable GPU for 10x speed:

1. Click "Settings" in the right panel
2. Under "Accelerator" select "GPU"
3. Click "Save"

With GPU: ~30-60 minutes total ⚡
Without GPU: ~3-5 hours 🐌
